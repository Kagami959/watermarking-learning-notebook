# The Stable Signature: Rooting Watermarks in Latent Diffusion Models
- **作者/年份/ venue**: Meta / 2023 / ICCV 2023
- **核心问题**: 如何在latent扩散模型中根植水印，使得生成的图像天然携带水印？
- **方法分类**: 图像水印 / 扩散模型
- **核心创新**: 通过微调LDM的解码器，使水印直接嵌入到生成过程中。不需要后处理修改，水印在解码阶段自然生成。
- **关键技术细节**:
  - 微调解码器：在LDM的解码器中嵌入水印编码能力
  - 生成过程：用户输入prompt → 生成latent → 解码器同时生成图像和水印
  - 检测：使用专用检测器从图像中提取水印
  - 与PAI的区别：需要微调模型（有训练成本），但水印更自然地融入生成过程
  - 与Tree-Ring的区别：在模型参数中而非噪声中嵌入水印
- **实验设置**:
  - 模型：Latent Diffusion Model
  - 评价指标：图像质量（FID）、水印检测率、不可感知性
  - 对比：与后处理水印方法对比
- **优点/局限**:
  - 优点：
    - 水印更自然地融入生成过程
    - 不需要后处理修改
    - 利用LDM的latent空间特性
  - 局限：
    - 需要微调解码器（训练成本）
    - 微调可能影响模型的通用生成能力
- **我的思考/疑问**:
  - 三种图像水印方案对比：
    - PAI：即插即用，无需训练，在provider端
    - Tree-Ring：即插即用，无需训练，在噪声空间
    - Stable Signature：需要微调，在模型参数中
  - 选择哪种方案取决于应用场景：PAI适合服务提供商，Tree-Ring适合快速部署，Stable Signature适合长期使用
