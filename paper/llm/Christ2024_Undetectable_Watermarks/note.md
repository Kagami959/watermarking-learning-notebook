# Undetectable Watermarks for Language Models (Christ et al., FOCS 2024)
- **作者/年份/ venue**: Christ, Gunn, Zamir / 2024 / FOCS 2024
- **核心问题**: 是否存在一种水印方案，在理论上保证不影响输出质量的同时仍然可检测？
- **方法分类**: 文本水印 / LLM水印 / 理论论文
- **核心创新**: 从理论上证明了不影响输出质量的水印是存在的。提出基于信息论安全的水印构造方法。
- **关键技术细节**:
  - 理论框架：信息论安全（information-theoretic security）
  - 核心结论：在一定条件下，可以构造出不可检测但可验证的水印
  - 数学工具：利用密码学中的伪随机性（pseudorandomness）和哈希函数
  - 与Goldwasser2024的区别：本论文的模型假设与Goldwasser不同，因此两者不矛盾
- **实验设置**:
  - 类型：理论证明为主，辅以实验验证
  - 评价指标：理论安全性保证 + 实际检测效果
- **优点/局限**:
  - 优点：
    - 提供了坚实的理论基础，证明不可检测水印在理论上可行
    - 为后续工程实现提供了理论指导
  - 局限：
    - 理论构造与实际工程实现之间存在gap
    - 效率和实用性需要进一步验证
- **我的思考/疑问**:
  - "不可检测水印"与"无失真水印不可能"这两篇论文看似矛盾，实际是因为模型假设不同
  - 本论文关注的是：水印不改变输出分布（不可检测）
  - Goldwasser2024关注的是：在某些条件下，无失真和可检测性之间存在矛盾
  - 理解两者的假设差异是理解水印理论边界的关键
