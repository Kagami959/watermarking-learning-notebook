# Distortion-Free Watermarks Are Not Possible (Goldwasser et al., 2024)
- **作者/年份/ venue**: Goldwasser, Kim, Vaikuntanathan / 2024 / （venue待补充）
- **核心问题**: 在某些条件下，无失真水印（不影响输出质量）和可检测性之间是否存在根本矛盾？
- **方法分类**: 文本水印 / LLM水印 / 理论论文（不可能性结果）
- **核心创新**: 证明了在特定模型假设下，无失真水印和可检测性之间存在根本性的不可能性矛盾。这是水印领域的关键理论极限。
- **关键技术细节**:
  - 不可能性证明：在一定条件下，任何水印方案要么会引入可察觉的失真，要么检测不可靠
  - 模型假设：与Christ2024的假设不同（这是两者结论不矛盾的原因）
  - 信息论工具：利用互信息和熵的不等式推导不可能性边界
- **实验设置**:
  - 类型：理论证明
  - 评价指标：不可能性定理
- **优点/局限**:
  - 优点：
    - 揭示了水印方案的根本理论极限
    - 帮助理解哪些条件下水印是可行的，哪些条件下不可行
    - 为工程实践提供了理论边界指导
  - 局限：
    - 不可能性结论依赖特定假设，实际场景中假设可能不成立
- **我的思考/疑问**:
  - 与Christ2024的关系：两者看似矛盾但实际不同——Christ证明"在假设A下不可检测水印存在"，Goldwasser证明"在假设B下无失真水印不可能"
  - 关键学习点：理论结果的价值在于揭示假设的重要性
  - 类似于音频水印中的容量-鲁棒性权衡：信息容量越大，鲁棒性越差
