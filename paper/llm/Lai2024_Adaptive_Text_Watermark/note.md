# Adaptive Text Watermark (Lai et al., 2024)
- **作者/年份/ venue**: Lai et al. / 2024 / （venue待补充）
- **核心问题**: Kirchenbauer2023的水印强度固定（统一偏置δ），未考虑不同token的上下文差异。如何根据token的语义重要性自适应调整水印强度？
- **方法分类**: 文本水印 / LLM水印
- **核心创新**: 根据token的上下文自适应调整水印强度——类似于音频水印中SilentCipher的心理声学掩蔽在文本领域的对应。
- **关键技术细节**:
  - 自适应机制：根据token的上下文语义重要性动态调整水印嵌入强度
  - 重要token（如关键实体词）：降低水印强度，保护文本质量
  - 非重要token（如常见虚词）：增强水印强度，提高检测置信度
  - 与Kirchenbauer2023的区别：从"统一δ"到"自适应δ(token)"
