# VideoSeal (Meta, 2025)
- **作者/年份/ venue**: Meta / 2025 / （venue待补充）
- **核心问题**: 从AudioSeal到视频水印，如何处理时序一致性对水印设计的影响？
- **方法分类**: 视频水印 / 深度学习
- **核心创新**: 将AudioSeal的水印框架扩展到视频领域，解决帧间一致性、时序冗余等视频特有的挑战。
- **关键技术细节**:
  - 从AudioSeal到VideoSeal的扩展：
    - 音频：处理1D时序信号
    - 视频：处理2D空间+1D时序信号
  - 视频特有挑战：
    - 帧间一致性：相邻帧的水印需要保持一致
    - 时序冗余：视频的连续帧高度相似，如何避免水印冲突
    - 编解码兼容性：视频压缩（H.264/H.265）对水印的破坏

- **我的思考/疑问**:
  - 视频水印比音频水印多了什么挑战？

    - 时序一致性：相邻帧的水印不能冲突
