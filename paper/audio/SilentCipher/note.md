# SilentCipher: Deep Audio Watermarking
- **作者/年份/ venue**: Sony / 2024.06 / arXiv
- **核心问题**: 在保证音频不可感知性的前提下，提升水印对常见音频压缩/编辑攻击的鲁棒性。
- **方法分类**: 深度学习
- **核心创新**: 提出心理声学掩蔽 + 可调SDR下限 + 伪可微压缩层 + 免感知损失训练的四合一水印框架。
- **关键技术细节**:
  - 嵌入/提取流程：
    1. **心理声学掩蔽**：让水印的相位与原始音频相反，并限制每个频率上的水印幅度不超过原始信号。水印能量集中在原始声音大的频段，利用人耳的"频率掩蔽效应"实现完全不可听。
    2. **可调的SDR下限**：引入用户控制的缩放因子α，直接规定水印强度与原始信号的能量比，使得最终信噪比（SDR）精确等于α。无需重新训练就能在推理时调节隐蔽性与鲁棒性之间的平衡。
    3. **伪可微压缩层**：针对MP3、AAC、OGG等不可微的压缩算法，前向传播时真正执行压缩，反向传播时让梯度"跳过"该层。使模型学会抵抗压缩攻击，同时避免梯度无法回传的问题。
    4. **免感知损失训练**：由于SDR已由α精确控制，不再需要额外的感知损失来约束失真。模型只优化消息分类的交叉熵损失，训练更简单、收敛更稳定。
  - 关键参数：缩放因子α（控制SDR）
