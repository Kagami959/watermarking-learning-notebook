# TriniMark: A Robust Generative Speech Watermarking Method for Trinity-Level Traceability
- **作者/年份/venue**: Yue Li, Weizhi Liu, Kaiqing Lin, Dongdong Lin, Kassem Kallas (华大/华师大/深大) / 2025 / arXiv:2504.20532
- **核心问题**: 现有语音水印通常只支持内容级溯源（验证是否为某模型生成），无法在模型被盗用或大规模部署时追溯到具体哪个用户泄露了生成内容。
- **方法分类**: 语音水印 / 扩散模型 / 溯源
- **核心创新**: 提出三联溯源（Trinity-Level Traceability）框架，同时支持**内容级**（水印信息可追溯）、**模型级**（可识别来源模型）、**用户级**（可识别生成请求的具体用户）三级溯源。
- **关键技术细节**:
  - **两阶段训练**：
    - **Stage I**：预训练时域水印编码器-解码器（带噪声层增强鲁棒性）
    - **Stage II**：波形引导微调(WGFT)，将水印能力注入扩散模型
  - **可变水印训练**：训练时使用不同水印消息，推理时可分配不同水印给不同用户
  - **架构**：轻量级时域编码器 + 时域门控卷积解码器(Temporal-aware Gated Convolutional Decoder)
  - **容量**：500 bps
- **实验设置**:
  - 模型：语音扩散模型
  - 评价指标：提取准确率、感知质量、溯源准确率
- **实验结果亮点**:
  - 同时支持三级溯源
  - 感知质量无损、高提取准确率
- **优点/局限**:
  - 优点：
    - 三联溯源：首个同时支持内容、模型、用户三级溯源的语音水印方法
    - 可扩展用户追踪：可变水印训练策略 → 推理时不同用户获得不同水印 → 大规模部署可行
    - 波形引导微调将预训练编解码器的水印能力有效迁移到扩散模型
  - 局限：
    - 两阶段训练流程较复杂
    - 需要为每个用户分配和管理唯一水印
- **我的思考/疑问**:
  - 与ZODiAC的对应关系：
    - ZODiAC对已有图像做DDIM反演→嵌入→再生成，只支持所有权验证（0-bit）
    - TriniMark两阶段训练将水印注入扩散生成过程，支持多bit信息，支持用户级溯源
  - 在实际部署中，用户级溯源需要管理大量唯一水印，可扩展性是一个关键挑战
