# SOLIDO: A Robust Watermarking Method for Speech Synthesis via Low-Rank Adaptation
- **作者/年份/venue**: Yue Li, Weizhi Liu, Dongdong Lin (华大) / 2025 / arXiv:2504.15035
- **核心问题**: 现有生成式语音水印方法需要训练完整编解码器或微调整个扩散模型，训练开销大。能否在不牺牲鲁棒性的前提下大幅降低训练成本？
- **方法分类**: 语音水印 / 扩散模型 / LoRA / 参数高效微调
- **核心创新**: 首次将LoRA（低秩适配）参数高效微调引入语音生成式水印，大幅降低训练开销的同时保持高鲁棒性。
- **关键技术细节**:
  - **LoRA微调**：冻结扩散模型原始参数，只训练两个低秩矩阵A和B（r << d, k）
  - **水印编码器**：仅三个操作（全连接+ReLU），将水印转换为扩散模型输入的潜变量
  - **水印解码器**：基于深度可分离卷积，可直接处理变长输入（无需截断或填充）
  - **语音驱动轻量微调策略(SDFT)**：联合优化语音质量和提取准确率
  - **容量**：2000 bps
- **实验设置**:
  - 模型：语音扩散模型
  - 基线：全参数微调(FPT)、参数冻结(PFT)等方法
  - 攻击类型：单一攻击、复合攻击（包括时间拉伸）
- **实验结果亮点**:
  - 单一攻击提取准确率 99.20%
  - 复合攻击提取准确率 98.43%
  - 抗时间拉伸比SOTA方法提升约23%
- **优点/局限**:
  - 优点：
    - LoRA水印：首个将参数高效微调(PEFT)用于语音水印的工作
    - 变长输入处理：深度可分离卷积解码器直接处理任意长度音频
    - 极高提取率：复合攻击下仍保持高准确率
    - 训练参数量大幅减少（对比全参数微调和参数冻结方法有明显优势）
  - 局限：
    - LoRA秩的选择影响水印容量和鲁棒性的平衡
    - 需要为每个模型训练LoRA权重
- **我的思考/疑问**:
  - 计算开销对比很有说服力：全参数微调(FPT)=100%参数量，参数冻结(PFT)=仅编解码器，SOLIDO(LoRA)=极少量低秩矩阵
  - 深度可分离卷积解码器能处理变长输入，这对实际部署中的裁剪和时间拉伸攻击很关键
  - 与GROOT对比：GROOT训练完整编解码器，SOLIDO用LoRA微调扩散模型本身，两条不同的技术路线
