# Smark: A Watermark for Text-to-Speech Diffusion Models via Discrete Wavelet Transform
- **作者/年份/venue**: Yichuan Zhang, Chengxin Li, Yujie Gu (九州大学) / 2025 / arXiv:2512.18791
- **核心问题**: 不同TTS扩散模型架构差异大，现有水印方法往往针对特定模型设计，缺乏通用性。能否做到一种水印方案适用于所有TTS扩散模型？
- **方法分类**: 音频水印 / TTS扩散模型 / 离散小波变换
- **核心创新**: 首个模型无关的通用水印方案，适用于所有TTS扩散模型。核心思路是利用离散小波变换(DWT)在反向扩散过程中将水印嵌入到听觉感知稳定的低频子带(LL)。
- **关键技术细节**:
  - **通用性来源**：所有TTS扩散模型共享相同的反向扩散数学范式（马尔可夫链去噪），Smark在这一层操作，不依赖特定模型架构
  - **DWT嵌入**：
    1. 对反向扩散过程中某时间步的梅尔频谱图做DWT → 分解为LL(低频近似)、LH、HL、HH四个子带
    2. 水印只嵌入LL子带（携带核心结构信息，抗干扰强）
    3. IDWT重构梅尔频谱图 → 继续反向扩散
  - **水印编解码器**：轻量级神经网络（全连接层+膨胀卷积+强度预测器）
  - **验证框架**：基于二项分布的假设检验，控制FPR/FNR
- **实验设置**:
  - 模型：多种TTS扩散模型
  - 基线：与特定模型绑定的水印方法对比
  - 评价指标：不可感知性、鲁棒性、通用性
- **实验结果亮点**:
  - 在多个不同的TTS扩散模型上均有效
- **优点/局限**:
  - 优点：
    - 模型无关：首次实现适用于所有TTS扩散模型的通用水印方案
    - DWT低频嵌入：利用DWT多分辨率分析能力，水印嵌入感知稳定区域
    - 自适应强度：动态预测嵌入强度，在感知掩蔽区域加强嵌入
    - 统计验证：假设检验框架确保统计可靠性
  - 局限：
    - 依赖DWT变换的数值稳定性
    - LL子带嵌入容量有限
