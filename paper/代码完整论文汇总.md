## 一、图像水印 (Image Watermarking)

### 1. Tree-Ring Watermarks (NeurIPS 2023)
- **论文内容**: 在扩散模型初始噪声的傅里叶域嵌入环形水印模式，通过 DDIM 反演检测水印。无需训练，即插即用。
- **代码仓库**: https://github.com/YuxinWenRick/tree-ring-watermark
- **完整性评估**: ✅ **完整**
  - `run_tree_ring_watermark.py` — 主实验运行脚本（支持多种攻击设置）
  - `run_tree_ring_watermark_fid.py` — FID 评估脚本
  - `scripts/tree_ring.sh` / `tree_ring_imagenet.sh` — 批处理实验脚本
  - PyPI 包 (`pip install tree-ring-watermark`)
  - 支持 CLIP Score 和 FID 计算

### 2. Stable Signature (ICCV 2023)
- **论文内容**: 通过微调 LDM 解码器将水印根植到潜扩散模型中，使所有生成图像天然携带水印。
- **代码仓库**: https://github.com/facebookresearch/stable_signature
- **完整性评估**: ✅ **完整**
  - `finetune_ldm_decoder.py` — 核心训练脚本
  - `run_eval.py` — 鲁棒性和质量评估脚本
  - `decode.ipynb` — Jupyter Notebook 交互式水印提取演示
  - `hidden/` — 完整的水印编解码器预训练代码
  - 在线 Demo: https://huggingface.co/spaces/imatge-upc/stable_signature

### 3. Gaussian Shading (CVPR 2024)
- **论文内容**: 将水印比特通过流密码加密，映射到服从标准高斯分布的潜变量，实现性能无损且无需训练的水印方案。
- **代码仓库**: https://github.com/zijin-yang/Gaussian-Shading (官方, 实际托管于 qwkkkkk)
- **完整性评估**: ✅ **完整**
  - `run_gaussian_shading.py` — 主实验脚本（支持水印嵌入/检测、CLIP Score、多种攻击）
  - `gaussian_shading_fid.py` — FID 评估脚本
  - 支持 SD 1.4/2.0/2.1
  - 可配置容量（默认 256 bits）、攻击参数、加密方式

## 二、音频水印 (Audio Watermarking)

### 4. AudioSeal (2024)
- **论文内容**: Meta 提出的局部化音频水印方案，支持 16-bit 消息，具有 SOTA 鲁棒性和极快检测速度。
- **代码仓库**: https://github.com/facebookresearch/audioseal
- **完整性评估**: ✅ **完整**
  - pip 安装 (`pip install audioseal`)
  - 完整的 Python API（Generator + Detector）
  - 预训练模型自动下载
  - README 提供完整使用示例
  - 支持 16-bit 秘密消息和逐样本检测

### 5. WavMark (2024)
- **论文内容**: 基于频谱图域的音频水印方案，使用同步比特串 + 水印比特串结构。
- **代码仓库**: https://github.com/wavmark/wavmark
- **完整性评估**: ✅ **完整**
  - pip 安装 (`pip install wavmark`)
  - 提供 encode/decode 完整示例代码
  - 社区 fork (NIkok0/Audio-wavmark) 提供额外的测试脚本和批处理工具
  - 在线 Demo: https://huggingface.co/spaces/M4869/WavMark
  - 音频样例: https://wavmark.github.io/

### 6. SilentCipher (Sony, 2024)
- **论文内容**: Sony 提出的深度学习音频水印方案，支持 16kHz 和 44.1kHz 采样率，40-bit 消息。
- **代码仓库**: https://github.com/sony/silentcipher
- **完整性评估**: ✅ **完整**
  - pip 安装 (`pip install silentcipher`)
  - 提供 encode/decode 完整示例
  - 包含 Colab 笔记本示例
  - 包含 Flask 服务器示例和 UI 前端展示
  - 预训练模型通过 HuggingFace 分发
  - HuggingFace: https://huggingface.co/sony/silentcipher

### 7. AudioMarkBench (NeurIPS 2024 Datasets & Benchmarks)
- **论文内容**: 系统的音频水印鲁棒性评测基准，涵盖 3 种 SOTA 水印方法、12 种无盒攻击、黑盒和白盒攻击。
- **代码仓库**: https://github.com/moyangkuo/AudioMarkBench
- **完整性评估**: ✅ **完整**
  - 完整的水印嵌入/检测流水线
  - 支持 AudioSeal、Timbre、WavMark 三种方法
  - 模拟无盒/黑盒/白盒攻击
  - 提供 AudioMarkData 数据集（基于 Common Voice，20,000 样本）
  - 包含公平性分析（语言、性别、年龄维度）

## 三、大语言模型水印 (LLM Watermarking)

### 8. Kirchenbauer LM-Watermarking (ICML 2023)
- **论文内容**: 首个面向 LLM 的生成式水印方案，通过软红名单策略在解码时偏置 token 选择嵌入水印。
- **代码仓库**: https://github.com/jwkirchenbauer/lm-watermarking
- **完整性评估**: ✅ **完整**
  - `demo_watermark.py` — Gradio 交互式 Demo + 最小可用示例
  - `app.py` — Gradio Web 界面
  - `watermark_processor.py` — 最小化 Watermark Processor/Detector 实现
  - `extended_watermark_processor.py` — 完整功能实现（推荐使用）
  - 支持 HuggingFace 模型
  - HF Demo: https://huggingface.co/spaces/tomg-group-umd/lm-watermarking

## 四、基础模型 (Foundation Models)

### 9. DDPM (NeurIPS 2020)
- **论文内容**: 去噪扩散概率模型（Denoising Diffusion Probabilistic Models），开创性提出扩散模型训练和采样框架。
- **代码仓库**: https://github.com/hojonathanho/diffusion
- **完整性评估**: ✅ **完整**（但有局限）
  - 包含完整的训练脚本 (`scripts/train.py`) 和评估脚本 (`scripts/evaluate.py`)
  - 包含 `GaussianDiffusion` / `GaussianDiffusion2` 完整实现
  - 支持多种损失类型、预测目标和方差处理方式
  - ⚠️ 基于 TensorFlow 1.15（较旧），需要特定环境配置

### 10. Latent Diffusion (CVPR 2022)
- **论文内容**: 潜扩散模型（LDM / Stable Diffusion 前身），在潜空间进行扩散过程，大幅降低计算成本。
- **代码仓库**: https://github.com/CompVis/latent-diffusion
- **完整性评估**: ✅ **完整**
  - `scripts/txt2img.py` — 文生图推理脚本
  - `scripts/sample_diffusion.py` — 无条件采样脚本
  - `scripts/inpaint.py` — 图像修补脚本
  - `scripts/knn2img.py` — 检索增强生成脚本
  - `main.py` — 训练入口（PyTorch Lightning）
  - YAML 配置系统，支持多种数据集和模型配置
  - 提供预训练模型下载脚本
